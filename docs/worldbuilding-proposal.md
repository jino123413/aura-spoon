# 기운 한 스푼 - 세계관 및 스토리라인 설계서

> A+B 하이브리드 방향: **기운 도감 + 마스코트 육성**

---

## 1. 세계 설정 (World Setting)

### 기운 마을 — "기운숲"

세상 어딘가, 사람들의 마음 한 켠에 **기운숲**이라는 작은 세계가 있습니다.

기운숲은 사람들이 서로에게 건네는 따뜻한 말, 진심 어린 인사, 조용한 응원 같은 **마음의 에너지**가 모여 만들어진 곳이에요. 이 숲에는 20가지 서로 다른 기운을 품은 작은 정령들 — **기운이**들이 살고 있습니다.

기운이들은 눈에 보이지 않지만, 사람의 **이름**을 통해 만날 수 있어요. 모든 이름에는 그날의 기운이 깃들어 있고, 이름을 불러주면 그 기운에 맞는 기운이가 기운숲에서 살짝 모습을 드러내죠.

### 기운숲의 풍경

기운숲은 계절 없이 늘 **은은한 빛**으로 가득한 곳이에요. 작고 둥근 빛 방울들이 공기 중에 떠다니고, 발밑에는 부드러운 이끼가 깔려 있어요. 기운이들은 저마다 좋아하는 장소가 있습니다:

- 숲 가운데에는 커다란 **기운나무**가 한 그루 서 있어요. 기운이들이 모여 쉬는 곳이죠.
- 기운나무 아래에는 **기운 샘**이 있어서, 사람들이 보내준 마음의 에너지가 샘물처럼 고여요.
- 기운이들은 이 샘물을 마시며 자라나고, 새로운 힘을 얻어요.

### 기운숲의 규칙

1. **하루에 한 번**, 이름을 부르면 기운이를 만날 수 있어요.
2. 기운이들은 **관심을 먹고 자라요**. 매일 찾아와 주는 것만으로도 기운이는 기뻐해요.
3. 모든 기운이는 **소중하고 동등해요**. 좋은 기운, 나쁜 기운은 없어요.
4. 기운이들은 서로 **어울리며** 살아요. 같이 있으면 더 빛나는 조합이 있죠.

## 2. 사용자-정령 관계 (User-Spirit Relationship)

### "기운 친구" — 돌보는 친구이자 서로 기운을 나누는 관계

사용자는 기운이의 **기운 친구**입니다. 단순한 주인이나 훈련사가 아니라, 서로에게 기운을 나누어 주는 **동등한 친구 관계**예요.

- 사용자가 매일 이름을 넣어 기운이를 만나면, 기운이는 그 관심을 에너지 삼아 자라나요.
- 동시에 기운이도 사용자에게 그날의 기운과 응원 메시지를 돌려줘요.
- 즉, **사용자가 기운이를 키우지만, 기운이도 사용자에게 기운을 줍니다.**

이 쌍방향 관계가 앱의 핵심 정서입니다:
> "내가 기운이에게 기운을 주고, 기운이가 나에게 기운을 줘요."

### 대표 기운이

사용자에게는 하나의 **대표 기운이**가 있어요. 이 기운이는 사용자가 가장 많이 만난(가장 자주 받은) 기운 타입에 해당하는 기운이입니다.

- 대표 기운이는 홈 화면에 크게 표시되며, 사용자의 분신이자 파트너 역할을 해요.
- 매일 기운을 주면 대표 기운이가 조금씩 성장하고, 진화 단계를 밟아 나가요.
- "나를 가장 닮은 기운이" 라는 의미를 가집니다.

### 관계의 변화

| 진화 레벨 | 관계 단계 | 설명 |
|-----------|----------|------|
| Lv.0 씨앗 | 첫 만남 | 기운숲에서 작은 씨앗을 발견했어요. 아직 누군지 모르지만, 왠지 끌려요. |
| Lv.1 새싹 | 인사 사이 | 기운이가 눈을 떴어요! 서로를 알아가기 시작했어요. |
| Lv.2 성장 | 친한 친구 | 기운이와 교감이 깊어졌어요. 서로의 기분을 느낄 수 있어요. |
| Lv.3 빛남 | 소울메이트 | 기운이가 빛나고 있어요. 서로에게 특별한 존재가 되었어요. |
| Lv.4 완성 | 하나의 기운 | 기운이가 완전체가 되었어요. 당신과 기운이는 이제 하나의 기운이에요. |

## 3. 20종 기운이 캐릭터 프로필

### 자연의 힘 계열 (원소형)

#### 1. 불꽃기운이 — 열정의 정령
- **성격**: 활발하고 에너지가 넘쳐요. 항상 무언가를 하고 싶어하고, 가만히 있으면 몸이 근질근질해요. 다른 기운이들 사이에서 분위기 메이커 역할.
- **말버릇**: "가보자고!" "해볼 만하잖아!"
- **좋아하는 것**: 새로운 도전, 달리기, 뜨거운 것
- **특징**: 불꽃 머리카락이 기분에 따라 활활 타오르거나 잔잔해져요.
- **관계**: 번개기운이와 베스트프렌드 (둘 다 빠르고 열정적). 노을기운이와 티격태격 (너무 느리다며).

#### 2. 오로라기운이 — 신비의 정령
- **성격**: 신비롭고 예측 불가능해요. 매번 다른 색깔, 다른 분위기를 풍겨서 다른 기운이들도 궁금해해요. 조용히 미소만 짓는 타입.
- **말버릇**: "글쎄, 오늘은 어떤 색일까?" "비밀이야~"
- **좋아하는 것**: 밤하늘, 색깔 바꾸기, 혼자만의 시간
- **특징**: 오로라 날개가 감정에 따라 색이 변해요.
- **관계**: 안개기운이와 마음이 잘 통해요 (둘 다 미스터리한 분위기). 크리스탈기운이가 늘 신기해하며 따라다녀요.

#### 3. 번개기운이 — 결단력의 정령
- **성격**: 생각보다 행동이 앞서는 타입이에요. 판단이 빠르고, 한번 정하면 곧장 실행해요. 의리 있고 듬직한 면도 있어요.
- **말버릇**: "지금이야!" "고민할 시간에 가자!"
- **좋아하는 것**: 퀴즈, 빠른 것, 번쩍이는 것
- **특징**: 지그재그 촉수가 흥분하면 전기가 찌릿찌릿 흘러요.
- **관계**: 불꽃기운이와 찰떡궁합 (열정+결단). 달님기운이와 팀을 이루면 완벽 (사고+실행).

#### 4. 해님기운이 — 따뜻함의 정령
- **성격**: 누구에게나 다정하고, 만나면 기분이 좋아지는 기운이예요. 말 한마디 한마디에 온기가 담겨 있어요. 기운숲의 인기쟁이.
- **말버릇**: "오늘도 좋은 하루야~" "네가 있어서 다행이야"
- **좋아하는 것**: 인사하기, 따뜻한 곳, 모두가 웃는 것
- **특징**: 태양 왕관이 늘 포근한 빛을 발해요.
- **관계**: 꽃잎기운이와 가장 친해요 (따뜻함+다정함). 눈꽃기운이와 서먹서먹 (너무 차갑다며).

#### 5. 달님기운이 — 지혜의 정령
- **성격**: 말이 적고 조용하지만, 한마디 할 때 핵심을 짚어요. 다른 기운이들의 고민 상담역. 밤에 가장 활발해지는 야행성.
- **말버릇**: "...그래서 네 마음은 어때?" "조금 더 생각해 볼래?"
- **좋아하는 것**: 고요한 밤, 별 구경, 깊은 대화
- **특징**: 초승달 귀걸이가 밤이 되면 더 밝게 빛나요.
- **관계**: 별빛기운이와 밤의 친구 (같이 하늘을 바라봐요). 번개기운이에게 종종 "좀 천천히 생각해"라고 조언해요.

#### 6. 바다기운이 — 포용력의 정령
- **성격**: 넓고 깊은 마음의 소유자. 어떤 기운이의 이야기도 끝까지 들어주고, 판단하지 않아요. 기운숲에서 가장 어른스러운 존재.
- **말버릇**: "괜찮아, 다 들어줄게" "네 탓이 아니야"
- **좋아하는 것**: 조개 줍기, 파도 소리, 모두를 안아주기
- **특징**: 몸에 파도 무늬가 감정에 따라 잔잔하거나 출렁거려요.
- **관계**: 숲기운이와 함께 기운숲의 힐러 콤비. 모든 기운이가 힘들 때 찾아가는 존재.

#### 7. 숲기운이 — 치유의 정령
- **성격**: 존재 자체가 위로가 되는 기운이. 함께 있으면 마음이 편해져요. 아프거나 지친 기운이를 돌봐주는 간호사 같은 역할.
- **말버릇**: "쉬어도 돼" "잠깐 여기 앉아 있자"
- **좋아하는 것**: 나뭇잎 모으기, 조용한 산책, 물 주기
- **특징**: 나뭇잎 머리장식에서 은은한 초록빛이 나며, 근처에 풀이 자라요.
- **관계**: 바다기운이와 힐러 콤비. 대지기운이와 자연 친구 (함께 있으면 기운숲이 더 푸르러져요).

#### 8. 바람기운이 — 자유의 정령
- **성격**: 어디에도 묶이지 않는 자유로운 영혼. 기운숲 이곳저곳을 돌아다니며 소식을 전해줘요. 장난기가 많고 호기심 넘침.
- **말버릇**: "심심한데 어디 갈까?" "여기 말고 저기도 가보자!"
- **좋아하는 것**: 여행, 높은 곳, 새로운 것
- **특징**: 바람개비가 기분에 따라 빠르게 또는 느리게 돌아요.
- **관계**: 벚꽃기운이와 놀기 좋은 사이 (같이 다니면 꽃잎이 날려요). 용암기운이와 정반대 성격이라 신기해해요.

### 하늘의 빛 계열 (천체형)

#### 9. 별빛기운이 — 영감의 정령
- **성격**: 항상 무언가를 상상하고 있어요. 다른 기운이들이 생각지 못한 아이디어를 뚝딱 내놓는 발명가 타입. 약간 엉뚱.
- **말버릇**: "이런 건 어때?" "상상해봐, 만약에..."
- **좋아하는 것**: 별 세기, 낙서, 엉뚱한 질문
- **특징**: 이마의 별이 좋은 아이디어가 떠오를 때 반짝 빛나요.
- **관계**: 달님기운이와 밤의 단짝. 무지개기운이와 함께하면 가장 신나는 조합 (상상+낙관).

#### 10. 크리스탈기운이 — 순수의 정령
- **성격**: 세상에서 가장 맑고 투명한 마음. 거짓말을 못 하고, 감정이 바로 얼굴에 드러나요. 순수해서 가끔 속기도 하지만, 그래서 더 사랑받아요.
- **말버릇**: "정말? 정말?" "와아, 예쁘다!"
- **좋아하는 것**: 빛나는 것, 투명한 것, 솔직한 대화
- **특징**: 크리스탈 뿔이 빛을 받으면 무지개빛으로 반사돼요.
- **관계**: 오로라기운이를 늘 신기해하며 졸졸 따라다녀요. 꽃잎기운이와 같이 있으면 세상에서 가장 순수한 공간이 만들어져요.

#### 11. 용암기운이 — 끈기의 정령
- **성격**: 느리지만 절대 멈추지 않아요. 묵묵하고 과묵한 편이지만, 한번 마음먹은 건 반드시 해내요. 든든한 맏형/맏언니 느낌.
- **말버릇**: "...천천히 가면 돼" "포기하지 마"
- **좋아하는 것**: 묵묵히 걷기, 바위 쌓기, 긴 낮잠
- **특징**: 바위 팔이 단단하지만, 만지면 의외로 따뜻해요.
- **관계**: 대지기운이와 묵묵한 우정 (말없이 옆에 앉아있기). 바람기운이가 답답해하지만 은근히 존경해요.

#### 12. 꽃잎기운이 — 다정의 정령
- **성격**: 섬세하고 다정해서 작은 변화도 놓치지 않아요. 다른 기운이의 기분을 누구보다 빨리 알아차리고, 조용히 곁에 다가가요.
- **말버릇**: "오늘 기분이 좀 다른 것 같아" "이거 네 거 줄까?"
- **좋아하는 것**: 꽃 키우기, 선물 주기, 다른 기운이 관찰하기
- **특징**: 볼에 있는 꽃잎 볼터치가 수줍을 때 더 붉어져요.
- **관계**: 해님기운이와 따뜻한 콤비. 크리스탈기운이와 순수한 친구. 천둥기운이의 숨은 따뜻함을 유일하게 알아차려요.

### 날씨와 계절 계열 (감정형)

#### 13. 무지개기운이 — 낙관의 정령
- **성격**: 어떤 상황에서도 밝은 면을 찾아내는 궁극의 낙천주의자. 웃음이 끊이질 않고, 슬픈 기운이를 보면 참지 못해요.
- **말버릇**: "괜찮아, 잘 될 거야!" "웃으면 복이 와!"
- **좋아하는 것**: 웃기, 노래, 비 갠 후 하늘
- **특징**: 무지개 꼬리가 기분 좋을 때 더 선명해져요.
- **관계**: 별빛기운이와 드림팀 (상상+낙관). 안개기운이가 우울할 때 찾아가는 존재.

#### 14. 안개기운이 — 감성의 정령
- **성격**: 풍부한 감성의 소유자. 음악을 들으면 금방 울고, 아름다운 것을 보면 눈물이 나요. 예술적이고 감수성이 깊어요.
- **말버릇**: "...이거 왜 이렇게 예뻐" "오늘 기분이 뭔가... 보랏빛이야"
- **좋아하는 것**: 음악, 시 짓기, 안개 낀 아침
- **특징**: 안개 옷이 감정에 따라 짙어지거나 옅어져요.
- **관계**: 오로라기운이와 미스터리 듀오. 무지개기운이가 항상 기분을 밝혀줘요.

#### 15. 눈꽃기운이 — 논리의 정령
- **성격**: 차분하고 냉철한 분석가. 감정보다 논리를 우선하지만, 차갑기만 한 건 아니에요. 정확하고 공정해서 기운이들 사이의 심판 역할.
- **말버릇**: "잠깐, 한번 정리해보자" "근거가 뭐야?"
- **좋아하는 것**: 퍼즐, 눈 내리는 풍경, 깔끔한 것
- **특징**: 눈결정 왕관이 집중할 때 선명하게 빛나요.
- **관계**: 해님기운이와 서먹서먹하지만 서로 필요한 사이. 달님기운이와 지적 대화 상대.

#### 16. 대지기운이 — 신뢰의 정령
- **성격**: 흔들림 없는 안정감의 상징. 약속을 반드시 지키고, 한결같아서 모두가 의지해요. 과묵하지만 행동으로 보여주는 타입.
- **말버릇**: "걱정 마, 내가 여기 있잖아" "약속은 약속이야"
- **좋아하는 것**: 같은 자리에 있기, 무거운 것 들어주기, 해 뜨는 것 보기
- **특징**: 산 모양 모자가 계절마다 조금씩 색이 변해요.
- **관계**: 용암기운이와 무언의 우정. 숲기운이와 자연 트리오. 바람기운이가 멀리 떠나면 묵묵히 기다려줘요.

#### 17. 벚꽃기운이 — 설렘의 정령
- **성격**: 항상 두근두근 설레는 마음을 안고 있어요. 새로운 것을 만나면 눈이 반짝이고, 특별한 순간을 누구보다 소중히 여겨요. 로맨티스트.
- **말버릇**: "두근두근! 뭔가 좋은 일이 생길 것 같아!" "이 순간 기억해 두자"
- **좋아하는 것**: 첫 만남, 꽃 구경, 설레는 것
- **특징**: 주변에 항상 꽃잎이 흩날리고, 설렐 때 더 많이 날려요.
- **관계**: 바람기운이와 함께 다니면 꽃잎 폭풍! 새벽기운이와 "새로운 시작" 동료.

#### 18. 천둥기운이 — 카리스마의 정령
- **성격**: 존재감이 강하고 리더십이 있어요. 겉은 강렬하지만 속은 다정한 츤데레 타입. 위기 상황에서 앞장서는 의리파.
- **말버릇**: "내가 앞에 서 있을게" "...별거 아니야 (사실 걱정함)"
- **좋아하는 것**: 리더 역할, 강한 것, (몰래) 귀여운 것
- **특징**: 몸의 번개 마크가 감정이 격해질 때 찌릿찌릿 빛나요.
- **관계**: 꽃잎기운이가 유일하게 속마음을 알아차려요. 불꽃기운이와 라이벌이자 동지.

#### 19. 노을기운이 — 여유의 정령
- **성격**: 느긋하고 여유로운 힐링 메이커. 서두르는 법이 없고, 모든 것에 "괜찮아~"를 붙여요. 함께 있으면 시간이 느리게 가는 것 같아요.
- **말버릇**: "괜찮아~ 천천히~" "뭘 그렇게 급해"
- **좋아하는 것**: 노을 보기, 낮잠, 느린 산책
- **특징**: 구름 스카프가 항상 부드럽게 나부껴요.
- **관계**: 불꽃기운이가 "너무 느리다"고 하지만, 결국엔 노을기운이 곁에서 쉬어가요. 새벽기운이와 하루의 시작과 끝을 함께해요.

#### 20. 새벽기운이 — 희망의 정령
- **성격**: 조용하지만 확실한 빛을 품고 있어요. 어둠 속에서도 내일을 믿는 힘이 있죠. 힘든 기운이에게 가장 먼저 다가가서 "괜찮을 거야"라고 말해주는 존재.
- **말버릇**: "새 날이 밝아오고 있어" "포기하지 마, 빛은 곧 와"
- **좋아하는 것**: 새벽 하늘, 첫 새소리, 조용한 기도
- **특징**: 머리 위의 새벽별이 어두울수록 더 밝게 빛나요.
- **관계**: 벚꽃기운이와 "새로운 시작" 동료. 노을기운이와 하루의 양 끝에서 서로를 바라봐요. 모든 기운이가 힘들 때 마지막으로 찾는 존재.

## 4. 스토리 진행 (Story Progression)

### 4-1. 첫 방문 (온보딩)

```

"안녕, 반가워요."

"이곳은 기운숲이에요.
사람들의 마음이 모여 만들어진 작은 세계죠."

[기운나무 아래에 작은 빛 한 점이 보인다]

"어? 저기 작은 빛이 보여요.
기운이 씨앗이 당신을 기다리고 있었나 봐요."

[씨앗이 살짝 흔들린다]

"이름을 알려주면,

```

**첫 번째 기운 결과 확인 후**:

```

"{기운이이름}이가 당신의 첫 번째 기운 친구가 되었어요."

[도감에 첫 번째 기운이가 등록되는 연출]

"기운숲에는 20가지 기운이가 살고 있어요.

"내일도 와줄 거죠?"
```

**이미 오늘 기운을 확인한 뒤 재접속**:

{기운이이름}이가 행복해하고 있네요."

## 5. 마스코트 육성 내러티브

### 대표 기운이 선정 방식

대표 기운이는 **자동**으로 정해집니다. 복잡한 선택 과정 없이, 사용자가 매일 기운을 확인하다 보면 자연스럽게 결정되어요.

**규칙**: 지금까지 가장 많이 받은 기운 타입 = 대표 기운이 타입

- 예: 10번 방문 중 불꽃기운 3회, 바다기운 2회, 기타... → 대표 기운이는 **불꽃기운이**
- 동률일 경우: 가장 최근에 받은 기운 타입이 우선
- 대표 기운이가 바뀔 수 있음 (자연스러운 전환)

**스토리적 해석**: "당신에게 가장 많이 찾아오는 기운이 = 당신과 가장 잘 맞는 기운이 = 당신의 대표 기운이"

### 대표 기운이 변경 시

```
"어라? 오늘부터 {새기운이}이가 당신의 대표 기운이가 되었어요!"

"예전 {이전기운이}이가 떠난 건 아니에요.
도감에서 언제든 만날 수 있어요."

"{새기운이}이가 당신에게 더 가깝게 다가온 거예요.
앞으로 잘 부탁해, 하고 인사하네요."
```

### 매일의 "기운 주기" 내러티브

기운 확인 행위를 **"기운 주기(먹이기)"** 로 표현합니다.

**흐름**:
1. 사용자가 이름을 입력한다
2. 오늘의 기운이 결정된다 (예: 바다기운)
3. 기운 오브가 생성되는 연출
4. 대표 기운이가 기운 오브를 받는다
5. 대표 기운이가 반응한다

**기운 주기 반응 (대표 기운이 타입별 개인화)**:

각 기운이가 기운을 받았을 때 자기 성격에 맞게 반응합니다:

| 대표 기운이 | 기운 받기 반응 (기본) |
|------------|---------------------|
| 불꽃 | "우와, 오늘 기운 맛있다! 힘이 불끈!" |
| 오로라 | "오늘의 기운은... 이런 색이구나. 예뻐." |
| 번개 | "짜릿해! 에너지 충전 완료!" |
| 해님 | "따뜻해~ 고마워, 오늘도 좋은 하루야!" |
| 달님 | "...고마워. (조용히 미소)" |
| 바다 | "감사히 잘 먹을게. 넉넉하게 받았어." |
| 숲 | "후~ 오늘도 좋은 기운이다. 고마워." |
| 바람 | "맛있다! 어디서 가져온 기운이야? 신기해!" |
| 별빛 | "오! 이 기운에서 뭔가 떠오르는데...?" |
| 크리스탈 | "와아! 반짝반짝해! 진짜 예쁘다!" |
| 용암 | "...냠. 든든하다. (묵묵히 씹는다)" |
| 꽃잎 | "오늘도 기운 가져와줬구나. 항상 고마워." |
| 무지개 | "야호! 오늘도 기운 받았다! 기분 좋아~!" |
| 안개 | "...이 기운, 오늘 나에게 잘 어울려." |
| 눈꽃 | "기운 영양 분석 완료. 양호해. 감사." |
| 대지 | "믿고 먹는 기운. 고마워, 약속 지켜줘서." |
| 벚꽃 | "두근두근! 오늘 기운은 뭔가 특별한 느낌!" |
| 천둥 | "흠, 나쁘지 않아. ...맛있어. (안 보임)" |
| 노을 | "고마워~ 천천히 먹을게, 급할 거 없잖아~" |
| 새벽 | "새로운 기운이 밝아오네. 오늘도 고마워." |

### 같은 기운 타입을 받았을 때 (보너스 반응)

대표 기운이와 같은 타입의 기운을 받았을 때 특별 반응:

```
"와! 이건 내 기운이잖아! 두 배로 맛있어!"
[경험치 보너스 표시]
```

### 기운 조합 반응 (선택적 확장)

특정 기운 조합에 따른 특별 반응:

```
대표: 불꽃기운이 + 오늘의 기운: 바다기운
→ "으... 물기운이다... 좀 차갑지만, 먹을 수는 있어!"

대표: 달님기운이 + 오늘의 기운: 해님기운
→ "해님기운... 밝아서 눈이 부시지만, 따뜻하네."

대표: 숲기운이 + 오늘의 기운: 바람기운
→ "바람이 불어오네~ 나뭇잎이 살랑살랑~"
```

## 7. 부록: 핵심 용어 정리

| 용어 | 의미 |
|------|------|
| 기운숲 | 기운이들이 사는 세계 |
| 기운이 | 20가지 기운을 품은 작은 정령 |
| 기운 친구 | 사용자와 기운이의 관계 |
| 대표 기운이 | 사용자가 가장 많이 만난 타입의 기운이 (홈 화면 파트너) |
| 기운 주기 | 매일 이름을 입력해 기운이에게 에너지를 주는 행위 |
| 기운 도감 | 20종 기운이의 수집 현황을 보여주는 도감 |
| 기운 마스터 | 20종을 모두 발견한 사용자의 칭호 |
| 기운나무 | 기운숲 중앙의 나무, 기운이들이 모이는 곳 |
| 기운 샘 | 기운나무 아래의 샘, 사용자의 관심 = 샘물 |
| 씨앗/새싹/성장/빛남/완성 | Lv.0~4 진화 단계의 이름 |

## 8. 설계 원칙 요약

1. **쌍방향 관계**: 사용자가 기운이를 키우고, 기운이도 사용자에게 기운을 준다. 일방적 소유가 아닌 우정.
2. **부담 없는 가벼움**: 매일 1회, 이름만 넣으면 끝. 못 와도 괜찮다는 분위기 (죄책감 유발 금지).
3. **모든 기운은 동등**: 좋은 기운/나쁜 기운이 없다. 점수화 금지. 모든 결과가 긍정적.
4. **자연스러운 발견**: 도감 수집이 보상이지만, 강요하지 않는다. 매일 하다 보면 자연스럽게 채워진다.
5. **귀엽고 따뜻하게**: 모든 텍스트는 친근하고 따뜻하게. 명령형보다 청유형. 존댓말(해요체) 사용.
6. **한국어 네이티브**: 번역체 금지. 자연스러운 한국어 구어체.
