# 기운 한 스푼 — 개편 방향 제안서

## 현재 상태 요약

- **3화면 구조**: 홈(이름 입력) → 리빌(2.5초 애니메이션) → 결과
- **20종 기운**: 이름+날짜 해시로 결정, 하루 1회
- **진화 시스템**: 컬렉션/스트릭 기반 Lv.0~4 (100장 이미지 생성 완료)
- **수집 요소**: 20종 도감, 연속 출석 스트릭
- **차별점**: 기운이 마스코트, 진화, 에너지 파트너

### 현재의 한계
1. 하루 1회 → 결과 확인 후 할 게 없음 (리텐션 약함)
2. 컬렉션이 숫자 게이지뿐 → 수집 동기 약함
3. 세계관/스토리 없음 → 감정적 몰입 부족
4. 진화 이미지 100장이 있지만 활용 맥락이 부족

## 방향 A: "기운 도감" — 수집형 도감 중심

### 컨셉
> "매일 이름을 넣어 새로운 기운이를 발견하고, 나만의 기운 도감을 완성하세요"

포켓몬 도감처럼, 20종의 기운이를 하나씩 발견하며 채워가는 **수집 완성형**. 각 기운이에 대한 상세 정보와 진화 단계를 열람할 수 있다.

### 세계관
- 세상에는 20종의 기운이 존재하며, 사람의 이름에는 그날의 기운이 깃든다
- 기운이는 관심을 받을수록 성장한다 (매일 방문 = 성장)
- 모든 기운이를 발견하면 "기운 마스터"가 된다

### 화면 구성 (4~5화면)

```
[홈] ─── 이름 입력 + 오늘의 기운이 (기존과 유사)
  │
  ├─ [리빌] ─── 리빌 애니메이션
  │      │
  │      └─ [결과] ─── 기운 상세 + 공유/리트라이
  │
  └─ [도감] ─── 20칸 그리드 (발견한 기운이 = 컬러, 미발견 = 실루엣)
         │
         └─ [기운 상세] ─── 탭: 기본정보 / 진화단계 / 호환기운
```

### UI 특징
- **도감 그리드**: 4×5 그리드, 발견된 기운이는 Lv.0 이미지로 표시, 미발견은 회색 실루엣+물음표
- **기운 상세**: 스와이프로 Lv.0→4 진화 단계 미리보기 (현재 내 레벨까지만 활성)
- **진화 프로그레스**: 각 기운이 카드에 진화 게이지 표시
- **마스터 배지**: 20종 완성 시 특별 UI 이펙트

### 스토리라인
```
Lv.0 "기운의 세계에 첫 발을 내딛었어요"
Lv.1 "기운이들이 당신을 알아보기 시작했어요"
Lv.2 "기운이들과 교감이 깊어지고 있어요"
Lv.3 "기운의 비밀에 가까워지고 있어요"
Lv.4 "축하해요! 모든 기운의 마스터가 되었어요"
```

### 장점
- 수집 동기가 명확 (20칸 채우기)
- 진화 이미지 100장 활용 극대화 (도감에서 모든 이미지 열람)
- 구현 복잡도 중간

### 단점
- 발견이 해시 기반이라 원하는 기운이를 노릴 수 없음 → 답답할 수 있음
- 도감 자체가 앱 내 체류를 늘리진 않음

### 추가 기능 화면 수: +2 (도감 목록, 기운 상세)

## 방향 B: "기운이 성장일기" — 마스코트 육성 중심

### 컨셉
> "매일 이름 기운을 모아 나만의 기운이를 키워보세요"

다마고치/하이펫처럼, **하나의 대표 기운이를 선택하여 매일 키운다.** 매일 기운을 주입하면 기운이가 반응하고 성장한다.

### 세계관
- 세상의 기운은 20가지 속성으로 나뉜다
- 당신만의 기운이 알이 있고, 매일 기운을 먹여 키울 수 있다
- 기운이는 주인이 어떤 기운을 주느냐에 따라 다르게 성장한다
- 연속으로 기운을 줄수록 기운이가 더 빨리 자란다 (스트릭 보너스)

### 화면 구성 (4~5화면)

```
[홈] ─── 나의 기운이 (중앙 대형 마스코트) + 성장 상태
  │       ├─ 기분 표시 (오늘 기운 미주입 시 시무룩)
  │       ├─ 레벨 + 경험치 바
  │       └─ "오늘의 기운 주기" 버튼
  │
  ├─ [이름 입력] ─── 이름으로 오늘의 기운 확인
  │      │
  │      └─ [먹이기 연출] ─── 기운이가 기운을 먹는 애니메이션
  │             │
  │             └─ [결과] ─── 기운 설명 + 성장 반영 + 공유
  │
  └─ [일기] ─── 캘린더형 기운 기록
         └─ 날짜별: 어떤 이름, 어떤 기운, 기운이 반응
```

### UI 특징
- **홈 대형 마스코트**: 화면 40% 차지, 터치하면 반응 (흔들흔들, 반짝)
- **기분 시스템**: 오늘 기운 미주입 시 졸린 표정 / 주입 후 행복 표정
- **먹이기 연출**: 기운 오브가 기운이에게 흡수되는 파티클 애니메이션
- **성장 일기**: 캘린더 뷰에 날짜별 기운 아이콘, 스트릭 시각화
- **진화 순간**: 레벨업 시 특별 연출 (화면 전체 플래시 + 새 모습 공개)

### 스토리라인
```
첫 방문: "기운이 알이 당신을 기다리고 있었어요. 매일 기운을 모아 알을 깨워주세요!"
Lv.0→1: "기운이가 알에서 깨어났어요! 눈이 반짝이네요."
Lv.1→2: "기운이가 팔을 뻗었어요! 주변을 탐색하기 시작했어요."
Lv.2→3: "날개가 돋아났어요! 기운이가 하늘을 동경하고 있어요."
Lv.3→4: "기운이가 완전체로 진화했어요! 왕관이 빛나요."
연속 스트릭 7일: "기운이가 당신과 함께하는 매일이 행복하대요 💜"
```

### 장점
- 감정적 몰입 최고 (내가 키우는 기운이)
- 리텐션 강함 (매일 안 키우면 시무룩)
- 진화 이미지 활용: 내 기운이의 현재 모습 = 내가 가장 많이 받은 기운 타입의 진화 이미지

### 단점
- "대표 기운이 선택" 로직 추가 필요 (가장 많이 받은 기운 타입?)
- 20종 이미지 중 자기 기운이 타입 외에는 볼 기회 적음
- 구현 복잡도 높음 (감정 시스템, 먹이기 연출, 캘린더)

### 추가 기능 화면 수: +2~3 (먹이기 연출, 일기/캘린더, 진화 이벤트)

## 방향 C: "기운 스토리" — 일일 스토리 중심

### 컨셉
> "매일 펼쳐지는 기운이의 이야기, 오늘은 어떤 기운의 이야기일까?"

매일 다른 **짧은 스토리/시나리오**가 등장하고, 기운 결과가 그날의 에피소드가 된다. 매거진/웹소설 형태.

### 세계관
- "기운 마을"에는 20명의 기운이 주민이 살고 있다
- 매일 마을에서 하나의 사건이 일어나고, 당신의 이름이 그 사건의 주인공을 결정한다
- 기운이들 사이에는 관계도가 있다 (친구, 라이벌, 짝꿍)
- 매일 방문할수록 마을의 이야기가 풍성해진다

### 화면 구성 (4화면)

```
[홈] ─── "기운 마을" 일러스트 + "오늘의 이야기 열기"
  │       └─ 현재 진화 레벨에 따라 마을 변화 (Lv.0 황무지 → Lv.4 번화한 마을)
  │
  ├─ [이름 입력] ─── 이름 입력 (기존과 동일)
  │      │
  │      └─ [결과 = 에피소드] ─── 오늘의 에피소드 카드
  │             ├─ 캐릭터: 기운이 이미지 (진화 레벨 반영)
  │             ├─ 에피소드 텍스트: 짧은 이야기 (3~4줄)
  │             ├─ 기운 키워드 + 한마디
  │             └─ 공유 카드 (에피소드 형태)
  │
  └─ [마을 주민록] ─── 발견한 기운이 캐릭터 목록
         └─ 각 기운이의 프로필 + 에피소드 기록
```

### UI 특징
- **마을 비주얼**: 홈 화면이 기운 마을 조감도 (진화 레벨별로 마을 성장)
- **에피소드 카드**: 결과가 카드 뉴스 / 웹소설 형태로 제공
- **관계도**: 기운이 사이의 관계를 시각적으로 표시 (에너지 파트너 확장)
- **마을 성장**: Lv.0 빈 땅 → Lv.1 텐트 → Lv.2 오두막 → Lv.3 마을 → Lv.4 왕국

### 에피소드 예시 (기운별 3개씩 = 60개 에피소드)
```
불꽃기운 에피소드 1:
"오늘 기운 마을에 불꽃기운이가 새로운 도전을 시작했어요.
{name}님처럼 망설이지 않고 한 발을 내딛었죠.
'실패해도 괜찮아, 도전 자체가 빛나는 거야!'
불꽃기운이의 따뜻한 불빛이 마을을 밝혔어요. 🔥"

오로라기운 에피소드 2:
"오늘 기운 마을의 하늘에 오로라가 펼쳐졌어요.
{name}님만의 색깔로 물든 하늘을 마을 주민들이 올려다봤죠.
'같은 하늘이어도 보는 사람마다 다른 빛이야.'
오로라기운이가 미소 지으며 말했어요. ✨"
```

### 장점
- 매일 새로운 콘텐츠 느낌 (같은 기운이라도 다른 에피소드)
- 공유 시 스토리가 있어 바이럴 효과 높음
- 세계관 몰입감

### 단점
- 에피소드 텍스트 대량 작성 필요 (최소 60개)
- 마을 일러스트 별도 에셋 필요
- 스토리 품질이 앱 전체 품질을 좌우

### 추가 기능 화면 수: +1~2 (마을 주민록, 에피소드 아카이브)

## 방향 D: "기운 카드" — 카드 수집/교환 중심

### 컨셉
> "매일 뽑는 기운 카드, 레어 카드를 모아 나만의 덱을 완성하세요"

트레이딩 카드 게임처럼, 기운이가 **등급이 있는 카드**로 제공된다. 진화 레벨 = 카드 등급.

### 세계관
- 기운이들은 카드 형태로 세상에 나타난다
- 같은 기운이라도 등급이 다르다 (일반 → 레어 → 에픽 → 레전더리 → 미스틱)
- 매일 1장의 카드를 뽑을 수 있고, 연속 출석하면 등급이 올라갈 확률이 높아진다
- 20종 × 5등급 = 100장의 카드를 모두 모으는 것이 최종 목표

### 화면 구성 (4화면)

```
[홈] ─── 카드 뽑기 연출 + 데일리 미션
  │       ├─ 카드팩 이미지 (두드리면 빛남)
  │       └─ "오늘의 카드 뽑기" 버튼
  │
  ├─ [카드 개봉] ─── 카드 뒤집기 애니메이션
  │      │       ├─ 등급별 다른 이펙트 (일반=심플, 레전더리=풀 이펙트)
  │      │       └─ NEW 뱃지 (첫 발견 시)
  │      │
  │      └─ [카드 상세] ─── 카드 앞뒤 + 기운 설명
  │
  └─ [카드 도감] ─── 100칸 그리드 (20종 × 5등급)
         ├─ 필터: 기운별 / 등급별
         └─ 카드 뒤집기: 앞면(이미지) ↔ 뒷면(설명)
```

### UI 특징
- **카드 디자인**: 기운이 이미지 + 등급 프레임 (일반=은색, 레어=금색, 에픽=보라, 레전더리=무지개, 미스틱=홀로그램)
- **개봉 연출**: 카드팩이 빛나면서 열리고, 카드가 뒤집히는 3D 플립
- **등급 이펙트**: 레전더리 이상은 파티클+글로우 배경
- **100칸 도감**: 5×20 그리드, 미획득은 실루엣
- **컬렉션 통계**: "보유 카드 47/100", "레전더리 3/20"

### 등급(진화) 획득 조건
```
일반 (Lv.0): 항상 획득 가능
레어 (Lv.1): 3일 연속 출석 후 확률 해금
에픽 (Lv.2): 7일 연속 출석 후 확률 해금
레전더리 (Lv.3): 14일 연속 출석 후 확률 해금
미스틱 (Lv.4): 20종 일반 모두 수집 후 확률 해금

획득 확률 (해금 후):
일반 50% / 레어 30% / 에픽 15% / 레전더리 4% / 미스틱 1%
```

### 장점
- 수집 동기 극강 (100장 완성, 레어 헌팅)
- 기존 100장 이미지 = 그대로 100장 카드
- 매일 뽑기 = 가챠 도파민
- 등급 시스템 = 자연스러운 리텐션

### 단점
- 가챠 요소 → 토스 심사에서 문제될 수 있음 (도박성?)
- 확률 시스템 구현 필요
- 기운의 "의미" 보다 등급에 집중될 수 있음
- 구현 복잡도 높음 (카드 3D 플립, 100칸 도감, 확률 시스템)

### 추가 기능 화면 수: +2~3 (카드 개봉, 카드 상세, 도감 그리드)

## 추천 조합

### 추천 1: A+B 하이브리드 — "기운 도감 + 마스코트 육성"
도감 수집의 명확한 목표 + 대표 기운이 육성의 감정적 몰입을 결합.
- 홈: 내 기운이 (가장 많이 받은 타입) + 도감 진행률
- 도감 탭에서 수집 현황 확인
- 구현 복잡도: 중~높음

### 추천 2: A+C 하이브리드 — "기운 도감 + 데일리 에피소드"
도감의 수집 골 + 매일 다른 에피소드의 콘텐츠 신선함 결합.
- 결과가 에피소드 카드 형태로 제공
- 도감에서 과거 에피소드 다시 읽기
- 구현 복잡도: 중

### 추천 3: D 단독 — "기운 카드 컬렉션"
100장 이미지를 가장 극대화하는 방향. 가챠 도파민 + 수집 완성.
- 토스 심사 리스크만 해결되면 가장 중독성 강함
- 구현 복잡도: 높음

## 공통 개선 (어떤 방향이든 적용)

1. **하단 탭 네비게이션**: 홈 / 결과 / 도감(또는 일기/카드) — 현재 단일 페이지를 다중 탭으로
2. **공유 카드 리디자인**: Instagram 스토리용 정사각형 카드 이미지 생성
3. **알림 푸시**: "기운이가 기다리고 있어요" / "새로운 기운이 등장!"
4. **다크모드**: 현재 라이트 전용 → 기운이 테마 다크모드 (별빛 배경)
5. **사운드**: 리빌 효과음, 진화 효과음 (선택적)
