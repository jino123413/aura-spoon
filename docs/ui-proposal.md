# 기운 한 스푼 — UI/UX 화면 구성 설계서

## 방향: A+B 하이브리드 "기운 도감 + 마스코트 육성"

---

## 1. 내비게이션 구조

### 하단 탭 바 (3탭)

현재 3화면 상태머신(home -> reveal -> result)을 **탭 기반 내비게이션 + 스택 전환**으로 확장한다.

```
[홈 탭]          [도감 탭]          [기운이 탭]
  |                |                  |
  home             encyclopedia       mascot
  |                |                  |
  reveal (stack)   detail (stack)     diary (stack)
  |                                   |
  result (stack)                      evolution-event (overlay)
```

- **하단 탭 바**: 3개 탭 (홈 / 도감 / 기운이)
- **스택 전환**: 탭 내부에서 화면 간 push/pop (reveal, result, detail 등)
- **오버레이**: 진화 이벤트 같은 특별 연출은 전체 화면 오버레이로 처리

### 탭 전환 규칙
- 탭 전환 시 각 탭의 마지막 상태 유지
- reveal/result는 홈 탭 내부 스택이므로 도감/기운이 탭으로 자유 이동 가능
- 토스 네비바 뒤로가기: 스택이 있으면 pop, 루트면 앱 종료

## 2. 화면별 상세 설계

### 2-1. 홈 탭

#### (A) 홈 화면 (기존 개편)

**목적**: 오늘의 기운 확인 입구. 대표 기운이를 중심으로 한 감정적 랜딩.

**핵심 변경**: 기존 진화 게이지 중심 -> "내 기운이"를 중앙에 크게 배치하여 육성감 강화.

```
┌─────────────────────────┐
│      기운 한 스푼        │ <- 토스 네비바 (고정 헤더)
│                         │
│    ╭─── ─── ─── ───╮    │
│    │   ◖ 🌱 ◗      │    │ <- 대표 기운이 (화면 35%)
│    │  (float anim)  │    │    터치 시 기운이 반응
│    │   Lv.1 새싹    │    │    진화 레벨에 따라 이미지 변화
│    ╰─── ─── ─── ───╯    │
│                         │
│  "기운이가 기지개를 켜고  │ <- 기운이 상태 멘트
│   있어요"               │    (시간대/스트릭/기분별 변화)
│                         │
│  ╭─ ─ ─ ─ ─ ─ ─ ─ ─╮   │
│  │ 🔥 연속 5일째      │   │ <- 스트릭 뱃지 (불꽃 메타포)
│  │ ●●●●● ○○          │   │    점 하나 = 1일, 7일 단위
│  ╰─ ─ ─ ─ ─ ─ ─ ─ ─╯   │
│                         │
│  ┌───────────────────┐  │
│  │ 이름을 입력하세요   │  │ <- 텍스트 인풋 (기존 유지)
│  └───────────────────┘  │
│                         │
│  [ 오늘의 기운 확인하기 ] │ <- CTA (기존 유지)
│                         │
│  ╭───────────────────╮  │
│  │ "오늘도 좋은 기운   │  │ <- 일일 명언 카드 (기존 유지)
│  │  가득하시길"        │  │
│  ╰───────────────────╯  │
│                         │
│  [홈]   [도감]   [기운이] │ <- 하단 탭 바
└─────────────────────────┘
```

**인터랙션**:
- 대표 기운이 터치: 짧은 흔들림(wiggle) + 말풍선 팝업 ("안녕!", "오늘도 기운 줘!" 등 랜덤)
- 오늘 이미 기운 확인 완료 시: CTA 대신 "오늘의 기운 다시 보기" 버튼 (결과 화면으로)
- 기운이 주변 파티클: 진화 레벨에 따라 수량/밝기 증가 (기존 particles-bg 활용)

**애니메이션**:
- 대표 기운이: `float` 3s + `pulseGlow` (기존 CSS 재활용)
- 터치 반응: `@keyframes wiggle` (좌우 흔들림 0.3s)
- 스트릭 뱃지: 진입 시 `popIn` 0.4s

**기운이 상태 멘트 시스템** (시간대별):
- 오전 6~11시: "좋은 아침이에요! 오늘의 기운이 기다리고 있어요"
- 오후 12~17시: "기운이가 같이 놀고 싶대요"
- 저녁 18~23시: "오늘 하루도 수고했어요"
- 새벽 0~5시: "기운이가 꿈나라에서 기다리고 있어요"
- 오늘 미방문 + 스트릭 유지 중: "기운이가 기다리고 있어요... 빨리 와주세요!"
- 오늘 이미 기운 확인: "오늘의 기운을 받았어요! 내일도 와주실 거죠?"

#### (B) 리빌 화면 (기존 유지 + 소폭 개선)

**목적**: 기운 확인 순간의 몰입감. 2.5초 애니메이션으로 기대감 조성.

```
┌─────────────────────────┐
│                         │
│                         │
│       ○   ○   ○         │ <- 3겹 동심원 링 (기존 유지)
│      ○  ●  ○            │    auraExpandRing 애니메이션
│       ○   ○   ○         │
│                         │
│   배경: #F8F5FF ->       │ <- 아우라색 그라디언트 전환
│         아우라색 15%     │
│                         │
│  "{name}"님의 기운은...   │ <- 1.2s 후 fadeIn (기존)
│        불꽃기운          │ <- 아우라 이름 (기존)
│                         │
│                         │
│  (탭바 숨김)             │ <- 리빌 중 탭바 숨김
└─────────────────────────┘
   (2.5초 후 자동 전환)
```

**변경점**:
- 리빌 중 하단 탭 바 숨김 (몰입감)
- 진화 레벨 높을수록 링 수 증가 (Lv.0: 2개, Lv.4: 5개)
- 결과 전환 시 기운이 에게 기운 흡수 파티클 연출 추가 (방향 B의 "먹이기 연출" 간소화 버전)

**애니메이션**: 기존 `auraExpandRing`, `textFadeIn` 100% 재활용.

#### (C) 결과 화면 (기존 개편)

**목적**: 오늘의 기운 상세 정보 + 공유 + 기운이 성장 반영.

```
┌─────────────────────────┐
│      기운 한 스푼        │ <- 헤더
│                         │
│  ╭────────────╮         │
│  │            │         │ <- 원형 오브: 기운이 + 아우라
│  │ 기운이     │         │    aura 테마색 글로우 (기존)
│  │ + 아우라   │         │    해당 기운의 진화 레벨 이미지
│  ╰────────────╯         │
│     ╭ 불꽃기운 ╮         │ <- 아우라 이름 뱃지 (기존)
│                         │
│  [활기 넘치는][도전적인]  │ <- 키워드 태그 (기존)
│  [에너지 폭발]           │
│                         │
│  "{name}님의 기운"       │ <- 부제 (기존)
│                         │
│  "열정이 넘치고..."      │ <- 성격 서술 (기존)
│                         │
│  ╭─ 기운이의 한마디 ─╮   │ <- 말풍선 (기존)
│  │ "오늘은 새로운...  │   │
│  ╰─── ─── ─── ───╯   │
│                         │
│  ╭─ 에너지 파트너 ──╮   │ <- 에너지 파트너 (기존)
│  │ 🌊 파도기운       │   │
│  ╰─── ─── ─── ───╯   │
│                         │
│  ┌─ 기운이 성장 현황 ─┐  │ <- *NEW* 성장 피드백 카드
│  │ +12 exp!           │  │    기운 주입 결과 시각화
│  │ ████████░░ Lv.1    │  │    경험치 애니메이션
│  │ "기운이가 기뻐하고  │  │
│  │  있어요!"          │  │
│  └────────────────────┘  │
│                         │
│  [ 친구에게 공유하기 ]    │ <- 공유 버튼 (기존)
│                         │
│  [AD] 다른 이름 기운 보기 │ <- 광고 버튼 (기존)
│                         │
│  [홈]   [도감]   [기운이] │ <- 하단 탭 바
└─────────────────────────┘
```

**새 요소 - 기운이 성장 피드백 카드**:
- 기운 확인 시 경험치 획득 (기본 10, 스트릭 보너스 +2/일)
- 경험치 바 fill 애니메이션 (0.6s ease-out)
- 레벨업 임계값 도달 시: 카드가 빛나며 "진화 가능!" 표시 -> 탭하면 진화 이벤트 오버레이

**변경점**:
- 결과 화면 하단에 "기운이 성장 현황" 카드 추가
- 공유 카드 리디자인 (Instagram 스토리 비율 지원)
- 에너지 파트너 카드: 탭하면 도감의 해당 기운이 상세로 이동

### 2-2. 도감 탭

#### (D) 도감 메인 화면

**목적**: 20종 기운이 수집 현황을 한눈에. 발견의 쾌감과 미발견의 호기심 자극.

```
┌─────────────────────────┐
│       기운 도감          │ <- 헤더
│                         │
│  ╭─── 수집 현황 ───╮    │
│  │                  │    │ <- 원형 프로그레스 (도넛 차트)
│  │    ╭──╮          │    │    발견 수 / 20
│  │   │7/20│         │    │    둘레가 채워지는 시각화
│  │    ╰──╯          │    │    "기운 탐험가" 칭호
│  │  기운 탐험가      │    │
│  ╰─── ─── ─── ───╯  │
│                         │
│  ┌────┬────┬────┬────┐  │
│  │ 🔥 │ 🌌 │ ⚡ │ ☀️ │  │ <- 4x5 그리드
│  │불꽃 │오로라│번개 │해님 │  │    발견: 컬러 이미지 + 이름
│  ├────┼────┼────┼────┤  │    미발견: 회색 실루엣 + "?"
│  │ 🌙 │ ?? │ ?? │ 🌊 │  │
│  │달님 │ ?  │ ?  │파도 │  │    각 칸 터치 -> 상세 화면
│  ├────┼────┼────┼────┤  │
│  │ ?? │ ?? │ 🌿 │ ?? │  │    미발견 칸도 터치 가능
│  │ ?  │ ?  │나뭇잎│ ?  │  │    -> "아직 발견하지 못한
│  ├────┼────┼────┼────┤  │       기운이에요" 토스트
│  │ ?? │ ?? │ ?? │ ?? │  │
│  │ ?  │ ?  │ ?  │ ?  │  │    *NEW* 마지막 발견 기운이에
│  ├────┼────┼────┼────┤  │    "NEW" 뱃지 표시
│  │ ?? │ ?? │ ?? │ ?? │  │
│  │ ?  │ ?  │ ?  │ ?  │  │
│  └────┴────┴────┴────┘  │
│                         │
│  [홈]   [도감]   [기운이] │ <- 하단 탭 바
└─────────────────────────┘
```

**인터랙션**:
- 그리드 칸 터치: 발견된 기운이 -> 상세 화면 push / 미발견 -> 토스트 메시지
- 프로그레스 도넛: 발견 시마다 둘레 fill 애니메이션
- 칭호 시스템 (숫자 대신 시각적 메타포):
  - 0~3종: "기운 새내기"
  - 4~8종: "기운 탐험가"
  - 9~14종: "기운 수집가"
  - 15~19종: "기운 현자"
  - 20종: "기운 마스터" (특별 이펙트)

**애니메이션**:
- 그리드 진입: 각 칸이 staggered `popIn` (0.05s 간격)
- 미발견 칸: 은은한 `shimmer` 효과 (호기심 유발)
- NEW 뱃지: `pulseGlow` 반복

**디자인 상세**:
- 그리드 칸: `88px x 88px`, `rounded-2xl`, 내부에 `56x56` 이미지
- 발견 칸: `bg-bg-card` + 아우라 테마색 `box-shadow` (미세 글로우)
- 미발견 칸: `bg-surface` + 내부 물음표는 `text-[#1A0B3E]/20`
- 그리드 gap: `12px`

#### (E) 기운이 상세 화면

**목적**: 개별 기운이의 모든 정보 열람. 진화 단계 미리보기로 성장 동기 부여.

```
┌─────────────────────────┐
│  < (뒤로)   불꽃기운     │ <- 헤더 (기운이 이름)
│                         │
│  ╭────────────────╮     │
│  │                │     │ <- 대형 이미지 (현재 진화 레벨)
│  │  기운이 이미지   │     │    `240x240`, 원형, 아우라 글로우
│  │  (Lv.2 버전)   │     │    배경에 기운 테마색 방사형
│  ╰────────────────╯     │
│                         │
│  불꽃기운  [열정]        │ <- 이름 + 에너지 칩
│                         │
│  [활기 넘치는] [도전적인] │ <- 키워드 태그
│  [에너지 폭발]           │
│                         │
│  "열정이 넘치고..."      │ <- 성격 서술
│                         │
│  ── 진화 단계 ──         │ <- 진화 스와이프 섹션
│                         │
│  ╭──╮  ╭──╮  ╭──╮       │
│  │Lv0│ │Lv1│ │Lv2│      │ <- 가로 스와이프 진화 카드
│  │씨앗│ │새싹│ │성장│     │    현재까지만 컬러
│  ╰──╯  ╰──╯  ╰──╯       │    미도달: 잠금 아이콘 + 흐림
│              ╭──╮  ╭──╮  │
│              │🔒│  │🔒│  │ <- Lv.3, Lv.4는 잠금
│              │빛남│ │완성│ │
│              ╰──╯  ╰──╯  │
│                         │
│  ╭─ 기운이의 한마디 ─╮   │ <- 말풍선
│  │ "오늘은 새로운...  │   │
│  ╰────────────────╯  │
│                         │
│  ╭─ 에너지 파트너 ──╮   │ <- 호환 기운이 카드
│  │ 🌊 파도기운       │   │    터치 -> 해당 기운이 상세
│  ╰────────────────╯  │
│                         │
│  [홈]   [도감]   [기운이] │ <- 탭 바
└─────────────────────────┘
```

**진화 스와이프 카드**:
- 가로 스크롤 가능한 카드 리스트 (snap scroll)
- 각 카드: `100x130`, 상단에 진화 이미지(`72x72`), 하단에 레벨명
- 현재 진화 레벨까지: 컬러 + 터치 시 확대 팝업
- 미도달 레벨: 반투명 + 잠금 아이콘 오버레이 + "Lv.{n}에 해금" 텍스트

**애니메이션**:
- 진입 시: 이미지 `popIn` 0.5s + 키워드 staggered `fadeIn`
- 진화 카드 스와이프: CSS `scroll-snap-type: x mandatory`

### 2-3. 기운이 탭

#### (F) 기운이 홈 화면 (마스코트 육성)

**목적**: "내 기운이"와의 교감. 육성의 중심. 감정적 유대 형성.

```
┌─────────────────────────┐
│       나의 기운이         │ <- 헤더
│                         │
│  ╭─────────────────╮    │
│  │                  │    │
│  │                  │    │
│  │    ◖  ◗         │    │ <- 대형 마스코트 (화면 40%)
│  │   (기운이)       │    │    대표 타입의 현재 진화 이미지
│  │    Lv.2         │    │    터치하면 반응 (흔들림+표정)
│  │                  │    │    주변에 기운 파티클
│  │                  │    │
│  ╰─────────────────╯    │
│                         │
│     "성장 기운이"        │ <- 현재 진화 단계명

│                         │
│  ╭── 성장 현황 ──╮       │
│  │ ●━━━━━━━━○━━━○ │       │ <- 다음 진화까지 진행 바
│  │ Lv.2     Lv.3 │       │    시작점 = 현재, 끝점 = 다음
│  │ 경험치 142/200 │       │    가득 차면 진화 이벤트
│  ╰─── ─── ─── ──╯       │
│                         │
│  ╭── 오늘의 기록 ──╮     │
│  │ 🔥 불꽃기운 받음  │     │ <- 오늘 받은 기운 (있을 때만)
│  │ +12 exp          │     │    없으면 "오늘의 기운을
│  ╰─── ─── ─── ───╯  │    │    아직 확인하지 않았어요"
│                         │
│  [ 기운 일기 보기 ]       │ <- 기운 일기 진입 버튼
│                         │
│  [홈]   [도감]   [기운이] │ <- 탭 바
└─────────────────────────┘
```

**대표 기운이 결정 로직**:
- 지금까지 가장 많이 받은 기운 타입 = 대표 기운이
- 동률 시 가장 최근에 받은 기운 타입 우선
- 대표 기운이의 진화 이미지를 표시

**기운이 감정 시스템**:
- 오늘 기운 미확인: 눈 반쯤 감김 (졸린 표정) -> 이미지에 CSS `opacity: 0.7` + `grayscale(20%)`
- 오늘 기운 확인 완료: 활기찬 표정 -> 풀 컬러 + 파티클 활발
- 스트릭 7일+: 특별 파티클 (하트 모양)
- 기존 float/pulseGlow 애니메이션을 감정 상태에 따라 속도/강도 조절

**인터랙션**:
- 마스코트 터치: `wiggle` 0.3s + 말풍선 랜덤 표시 (1.5초 후 사라짐)
  - 기운 미확인: "기운 주세요...", "졸려요..."
  - 기운 확인 후: "고마워요!", "기운이 넘쳐요!", "내일도 와줄 거죠?"
- 마스코트 길게 누르기(0.5s): 쓰다듬기 연출 (파티클 burst)

**애니메이션**:
- 대형 마스코트: `float` 2.5s (기존보다 약간 빠름) + 레벨별 글로우
- 경험치 바: 진입 시 0% -> 현재값으로 fill (0.8s ease-out)
- 진화 이벤트 트리거: 바 100% 도달 시 `@keyframes shakeAndGlow` -> 오버레이 전환

#### (H) 진화 이벤트 오버레이

**목적**: 진화 순간의 특별한 경험. 달성감 극대화.

```
┌─────────────────────────┐
│ (전체 화면 오버레이)      │
│ 배경: 반투명 다크 + 파티클│
│                         │
│                         │
│     ╭─────────╮         │
│     │ Lv.1    │         │ <- 이전 진화 이미지 (작게)
│     │ 새싹    │         │
│     ╰────┬────╯         │
│          │              │
│        ╲│╱             │ <- 빛 파티클 방사
│     ╭─────────╮         │
│     │         │         │
│     │ Lv.2    │         │ <- 새 진화 이미지 (크게, 글로우)
│     │ 성장    │         │    스케일 0 -> 1.1 -> 1.0
│     │         │         │
│     ╰─────────╯         │
│                         │
│  "기운이가 날개를 펼쳤어요!" │ <- 진화 메시지
│                         │
│  [ 확인 ]                │ <- 닫기 버튼
│                         │
└─────────────────────────┘
```

**연출 시퀀스** (총 3초):
1. **0~0.5s**: 배경 어두워짐 + 이전 이미지 중앙 배치
2. **0.5~1.5s**: 이전 이미지 축소 + 빛 파티클 폭발 (방사형)
3. **1.5~2.5s**: 새 이미지 `scale(0)` -> `scale(1.1)` -> `scale(1.0)` + 글로우 확대
4. **2.5~3.0s**: 진화 메시지 `fadeIn` + 확인 버튼 표시

**파티클 효과**:
- 중앙에서 방사형으로 퍼지는 별 모양 파티클 20개
- 기운이 테마색 기반
- CSS `@keyframes particleBurst` (translateX/Y + opacity + scale)

## 5. 진화 시각화 상세

### 진화 경험치 시스템

```
레벨    필요 경험치    누적      해금 조건 (OR)
──────────────────────────────────────────
Lv.0    -            0         시작
Lv.1    50           50        3종 발견 OR 3일 연속
Lv.2    100          150       8종 발견 OR 7일 연속
Lv.3    200          350       14종 발견 OR 14일 연속
Lv.4    350          700       20종 발견
```

경험치 획득:
- 기본: 매일 기운 확인 시 +10
- 스트릭 보너스: 연속 n일 시 +(n*2) 추가 (최대 +14)
- 신규 기운 발견: +20 보너스

### 진화 진행 시각화 (기운이 탭)

- 가로 진행 바: `height: 8px`, `rounded-full`
- 시작점: 현재 레벨 마커 (컬러 도트)
- 끝점: 다음 레벨 마커 (회색 도트)
- fill: `linear-gradient(90deg, var(--primary-light), var(--primary))`
- 현재값 위치에 작은 빛나는 도트 (indicator)

### 진화 이미지 활용

기존 100장 이미지(20종 x 5레벨)를 다음과 같이 활용:
- **도감 그리드**: 각 기운이의 Lv.0 이미지
- **도감 상세 진화 카드**: 해당 기운이의 Lv.0~4 이미지 (해금된 것만 컬러)
- **결과 화면 오브**: 해당 기운의 현재 진화 레벨 이미지
- **기운이 탭 대형 마스코트**: 대표 기운이의 현재 진화 레벨 이미지
- **진화 이벤트**: before/after 이미지 비교
