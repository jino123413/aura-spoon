## 네비게이션 아키텍처

```
┌─────────────────────────────────────────────────┐
│                   AppProvider                    │
│            (useReducer + Context)                │
│                                                  │
│  ┌─────────────────────────────────────────┐    │
│  │              AppInner                    │    │
│  │                                          │    │
│  │  ┌──────┐  ┌──────────┐  ┌───────────┐ │    │
│  │  │ 홈   │  │ 도감     │  │ 기운이    │ │    │
│  │  │ Tab  │  │ Tab      │  │ Tab       │ │    │
│  │  └──┬───┘  └────┬─────┘  └─────┬─────┘ │    │

│  │  ┌──────────── Overlays ──────────────┐ │    │
│  │  │  reveal → result → aura-detail     │ │    │
│  │  │           evolution-event          │ │    │

│  │  └────────────────────────────────────┘ │    │

└─────────────────────────────────────────────────┘
```

## 화면 플로우

```
[첫 방문] ─→ Home(이름 입력) ─→ Reveal(2.5s) ─→ Result(스크롤)
                                                     │
                                                     ├─→ "공유하기"
                                                     ├─→ "AD 다른 이름" ─→ Home
                                                     └─→ 진화 조건 충족 시 ─→ EvolutionOverlay

[재방문] ─→ Home("오늘의 기운 다시 보기") ─→ Result

[탭 이동]
  홈 ←→ 도감 ←→ 기운이
              │
              └─→ 셀 탭 ─→ AuraDetailScreen (overlay z-40)
```

## 배경 이미지 필요 여부

### 결론: 배경 이미지 불필요

| 항목 | 현재 구현 | 배경 이미지 대안 | 판단 |
|------|----------|----------------|------|
| 전체 배경 | `#F8F5FF` 단색 라벤더 | 패턴/텍스처 | **불필요** — 파티클이 이미 생동감 제공 |
| 파티클 레이어 | CSS `floating-particle` 8~30개 | 없음 | 이미 충분한 시각적 레이어링 |
| Reveal 배경 | auraColor 15% 그라디언트 | — | 동적이라 이미지 불가 |
| Result 배경 | `var(--bg)` + orb glow | — | 오브 글로우가 배경 역할 |
| Evolution | 반투명 다크 오버레이 | — | 파티클 폭발이 배경 역할 |

**이유:**
1. **파티클 시스템이 배경 역할** — 레벨별로 8→30개 증가, 이미 "살아있는 느낌"
2. **퍼포먼스** — 배경 이미지는 메모리 추가 사용, 토스 미니앱은 경량 유지 필수
3. **색상 통일감** — `#F8F5FF` 단색이 아우라별 themeColor와 대비 역할
4. **번들 사이즈** — 현재 CSS 28.3KB, 이미지 추가 시 불필요한 증가

**만약 추가한다면?**
- 기운이 탭(MascotScreen)에만 미묘한 radial-gradient 배경 고려 가능
- `radial-gradient(ellipse at center top, ${mascotColor}08 0%, transparent 60%)` 정도
- 이미지 파일이 아닌 CSS gradient로 충분

## 앱 아이콘 제안

### 디자인 철학 (Signal Geometry 적용)

현재 앱: "기운" = 에너지 정령, "한 스푼" = 떠먹는 행위
핵심 기호: **빛 구슬 + 스푼**

### 안 A: 빛 구슬 in 스푼 (추천)
```
┌──────────────────┐
│                   │
│     ╭──────╮      │
│    ╱  ◉    ╲     │  ← 빛나는 구슬 (primary gradient)
│   │  ◉◉◉  │     │     #7B61FF → #B4A0FF
│   │   ◉◉   │     │     중심에서 방사형 글로우
│    ╲      ╱      │
│     ╰──┬──╯      │
│        │          │  ← 스푼 손잡이 (얇은 곡선)
│        │          │     #E8DEFF 또는 white
│        ╰          │
│                   │
│  배경: #7B61FF    │  ← primary 단색
│  (또는 gradient)   │     #5A3FD6 → #9D8AFF
│                   │
└──────────────────┘
```
- **콘셉트**: 스푼 위에 빛나는 에너지 구슬
- **Signal Geometry**: 원(구슬) + 곡선(스푼) = 2개 요소로 최대 상징
- **색상**: primary purple gradient, 구슬에 white glow
- **강점**: 앱 이름("한 스푼")과 직관적 연결

### 안 B: 에너지 파동 원
```
┌──────────────────┐
│                   │
│      ╭────╮       │
│    ╱  ╭──╮  ╲    │  ← 3겹 동심원 (Reveal 모티프)
│   │  │ ● │  │    │     바깥: #B4A0FF 20%
│    ╲ ╰──╯  ╱     │     중간: #9D8AFF 40%
│      ╰────╯       │     중심: #7B61FF (solid dot)
│                   │
│  배경: #1A0B3E    │  ← 다크 퍼플 (대비 극대)
│  (또는 gradient)   │
│                   │
└──────────────────┘
```
- **콘셉트**: 기운이 퍼져나가는 파동
- **앱 내 Reveal 애니메이션과 시각적 일관성**
- **색상**: 다크 배경에 밝은 동심원 = 빛이 퍼지는 느낌
- **강점**: 미니멀, 토스 앱 목록에서 눈에 띔

### 안 C: 기운이 캐릭터 얼굴
```
┌──────────────────┐
│                   │
│      ╭─────╮      │
│     ╱       ╲     │  ← 기운이 얼굴 (반투명 글로우)
│    │  ◕  ◕ │     │     Lv.2~3 수준의 귀여운 형태
│    │   ▽   │     │     단순화된 눈 + 미소
│     ╲     ╱      │
│      ╰───╯       │
│        ✨         │  ← 작은 스파클 1~2개
│                   │
│  배경: gradient    │  ← #7B61FF → #E8DEFF
│                   │
└──────────────────┘
```
- **콘셉트**: 기운이 마스코트 직접 활용
- **강점**: 캐릭터 인지도 → 앱 재방문 유도
- **약점**: 작은 사이즈에서 디테일 소실 가능

### 추천 순위

| 순위 | 안 | 이유 |
|------|-----|------|
| **1** | **A (빛 구슬 in 스푼)** | 앱 이름과 1:1 매칭, Signal Geometry 원칙 부합, 기억 잔존율 높음 |
| 2 | B (에너지 파동) | 미니멀하고 세련됨, 다크 배경으로 토스 목록에서 차별화 |
| 3 | C (기운이 얼굴) | 친근하지만 아이콘 크기에서 가독성 문제 가능 |
